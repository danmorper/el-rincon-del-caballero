Convolutional Neural Networks

The only use of Convolutional Neural Networks in my project is for Image Analysis.

Structure of an image

A black and white image can be seen as a matrix in which each element of the matrix corresponds to a pixel in the image and its value represents usually the intensity on a scale from 0 (black) to 255 (white).

However, since the 1950s we are used to see color images, thus we require RGB (Red, Green, Blue) images. These are what we call channels and they work exactly the same as the black and white we previously explained.

  • Each pixel is represented by an element in the matrix
  • Values of the matrix’ elements range from 1 to 255 depending on the intensity

Therefore, when we deal with image analysis we actually have three matrices
